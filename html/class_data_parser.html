Opis szczegółowy

Odpowiada za konwersję danych w formacie JSON (jako QByteArray) na struktury C++ zdefiniowane w DataStructures.h.

Klasa udostępnia metody do parsowania różnych typów odpowiedzi z API GIOS: listy stacji, listy czujników, danych pomiarowych oraz indeksu jakości powietrza. Zawiera również prywatne metody pomocnicze do bezpiecznego odczytywania wartości różnych typów z obiektów QJsonObject.

◆ parseAirQualityIndex()

AirQualityIndex DataParser::parseAirQualityIndex ( const QByteArray & jsonData)

Parsuje dane JSON zawierające indeks jakości powietrza (AQI) dla danej stacji.

Parametry
jsonDataDane w formacie JSON jako QByteArray. Oczekiwany obiekt indeksu AQI.
Zwraca
Obiekt AirQualityIndex. Zwraca obiekt z stationId = -1 w przypadku błędu parsowania JSON lub gdy obiekt JSON nie zawiera poprawnego ID stacji.
Nota
Poprawnie obsługuje brakujące lub null'owe pola dla poszczególnych poziomów indeksów, ustawiając dla nich wartości domyślne (ID=-1, Name="N/A").

◆ parseSensorData()

SensorData DataParser::parseSensorData ( const QByteArray & jsonData)

Parsuje dane JSON zawierające dane pomiarowe (serię czasową) dla danego czujnika.

Parametry
jsonDataDane w formacie JSON jako QByteArray. Oczekiwany obiekt zawierający klucz 'key' i tablicę 'values'.
Zwraca
Obiekt SensorData. Zwraca obiekt z pustym kluczem i/lub pustym wektorem wartości w przypadku błędu parsowania JSON lub niekompletnych danych.
Nota
Poprawnie obsługuje wartości 'null' w pomiarach, konwertując je na NaN.

Parametry
